# 单词

# 词根-走

# process

n.过程、进程；v.处理、加工

# processor

n.处理器

# procession

n.队伍，行列；一列，一排；列队行进

# graduate

v.毕业；n.（本科）毕业生

# undergraduate

n.大学肄业生

# ingredient

n.原料、要素、组成部分

# aggressive

a.侵略性的、激进的

# aggression

n.进攻、侵略

# aggress

v.侵略

# progress

v/n.前进、进步

# progressive

n.取得了很大进步的

# proceed

v.前进、继续进行；继续

# proceedings

n.诉讼、议程

# procedure

n.程序、步骤

# precede

v.领先于、优于

# precedent

n.先例、前例；a.先前的

# precedented

a.有过先例的

# unprecedented

a.史无前例的、空前

# precedence

n.优先、居先

# priority

n.优先权、优先

# access

v/n.接近、进入；使用、获取

# accessible

a.可接近的、可使用的、可获得

# accessory

n.附件、配件、从犯

# recession

n.衰退；凹处；后退；不景气

# recede

v.后退；撤退

# excess

n.无节制；过度，过量；超过，超额

# excessive

adj.过分的；过多的，极度的

# exceed

vt.胜过；超过

# exceedingly

adv.非常；极其；极度地

# cease

vt/vi/n.停止；终了

# ceaseless

vt/vi/n.停不下来的，永无休止的
